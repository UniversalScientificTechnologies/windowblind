AROM: autonomous robotic observatory manager

AROM je set malých programů postavených na systému ROS. AROM je software původně určený pro řízení astronomických dalekohledů a příslušenství observatoří. Díky své univerzálnosti lze AROM použít i na jiných systémech.

Tento systém je vytvořen s hardwarem stavebnice MLAB.
